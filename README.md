# URT Portfolio Hub

Prototype task management / portfolio management berbasis React + Vite + Tailwind.

## Cara menjalankan

1. Install Node.js LTS
2. Buka folder ini di VS Code
3. Jalankan:

```bash
npm install
npm run dev
```

4. Buka URL yang muncul di terminal, biasanya:

```bash
http://localhost:5173
```

## Fitur utama

- Dashboard summary
- Projects list
- Board ala Kanban / Asana
- Goals view
- Timeline view
- Edit progress langsung
- Checklist / subtasks
- Tambah project baru
- Penyimpanan lokal di browser (localStorage)

## Catatan

Ini masih frontend prototype. Belum ada backend, login, database, audit trail, atau sinkronisasi multi-user.
